# Zenkai Anime

Zenkai Anime is a modern anime streaming web application built with React, Tailwind CSS, Firebase, and a custom Node.js backend. It features responsive design, mobile app compilation via Capacitor, and seamless playback.

## Features
- **Modern UI/UX**: Beautiful glassmorphism, fluid animations, and a rich dark mode.
- **Anime Catalog**: Search, filter by genres, and track popular anime.
- **User Accounts**: Login, favorites, watch history, and sync across devices via Firebase.
- **Cross-Platform**: Deployable as a web app (Vercel) and native mobile app (Android via Capacitor).
- **Custom Video Player**: Built-in player with skip/rewind, playback speed, and quality controls.
